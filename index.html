Privacy Policy / 개인정보 처리방침

최종 업데이트: 2026년 1월

1. 수집하는 정보

Ringinout은 사용자의 개인정보를 수집하지 않습니다.

  • 위치 정보: 알람 기능을 위해 기기 내에서만 처리되며, 외부 서버로 전송되지 않습니다.
  • 음성 인식: 알람 입력을 위해 일시적으로 마이크를 사용하며, 음성 데이터는 서버에 저장되지 않습니다.
  • 계정 정보: Google 로그인 시 이메일 주소는 익명화된 랜덤 ID로 변환되어 저장됩니다. 원본 이메일은 저장되지 않습니다.

2. 익명화된 ID 사용 목적

익명화된 ID는 오직 유료 구독 상태 확인 목적으로만 사용됩니다. 이 ID를 통해 개인을 식별하거나 추적할 수 없습니다.

3. 데이터 저장

모든 알람 데이터와 장소 정보는 사용자의 기기 내에만 저장되며, 외부 서버로 전송되지 않습니다.

4. 제3자 공유

Ringinout은 어떠한 사용자 정보도 제3자와 공유하지 않습니다.

5. 사용 권한 안내

  • 위치 권한 (백그라운드 포함): 지정 장소 도착/출발 감지 알람 기능에 사용됩니다.
  • 마이크 권한 (RECORD_AUDIO): 음성으로 알람 시간 및 장소를 입력할 때만 사용됩니다. 녹음 내용은 저장되거나 전송되지 않습니다.
  • 알림 권한: 알람 발생 시 사용자에게 알림을 전달하기 위해 필요합니다.

6. 문의

개인정보 관련 문의사항이 있으시면 앱 내 '건의사항 보내기' 기능을 이용해주세요.

1. Information We Collect

Ringinout does not collect personal information.

  • Location data: Processed only on your device for alarm functionality. Not sent to external servers.
  • Voice recognition: The microphone is used temporarily to enter alarm details via voice. Audio data is not stored on any server.
  • Account info: When signing in with Google, your email is converted to an anonymized random ID. The original email is not stored.

2. Purpose of Anonymized ID

The anonymized ID is used solely to verify premium subscription status. This ID cannot be used to identify or track individuals.

3. Data Storage

All alarm and location data is stored only on your device and is not transmitted to external servers.

4. Third-Party Sharing

Ringinout does not share any user information with third parties.

5. Permissions Explained

  • Location (including background): Used to detect arrival/departure at designated places for alarm triggering.
  • Microphone (RECORD_AUDIO): Used only when you choose to input alarm time or location by voice. Recordings are never stored or transmitted.
  • Notifications: Required to deliver alarm alerts to the user.

6. Contact

For privacy-related inquiries, please use the 'Send Feedback' feature in the app.

1. 収集する情報

Ringinoutはユーザーの個人情報を収集しません。

  • 位置情報：アラーム機能のためにデバイス内でのみ処理され、外部サーバーには送信されません。
  • 音声認識：アラームの入力時に一時的にマイクを使用します。音声データはサーバーに保存されません。
  • アカウント情報：Googleログイン時、メールアドレスは匿名化されたランダムIDに変換されます。元のメールアドレスは保存されません。

2. 匿名化IDの使用目的

匿名化IDは、有料サブスクリプションの確認のみに使用されます。このIDによって個人を特定・追跡することはできません。

3. データの保存

すべてのアラームデータと場所情報はユーザーのデバイス内にのみ保存され、外部サーバーには送信されません。

4. 第三者への共有

Ringinoutはいかなるユーザー情報も第三者と共有しません。

5. 権限の説明

  • 位置情報（バックグラウンド含む）：指定した場所への到着・出発を検知するアラーム機能に使用されます。
  • マイク（RECORD_AUDIO）：音声でアラームを入力する場合のみ使用されます。録音内容は保存・送信されません。
  • 通知：アラームをユーザーに届けるために必要です。

6. お問い合わせ

プライバシーに関するお問い合わせは、アプリ内の「フィードバックを送る」機能をご利用ください。

1. 收集的信息

Ringinout不收集用户个人信息。

  • 位置信息：仅在设备内处理，用于闹钟功能，不发送至外部服务器。
  • 语音识别：输入闹钟时临时使用麦克风，语音数据不会存储在任何服务器上。
  • 账户信息：通过Google登录时，邮箱地址将被转换为匿名随机ID。原始邮箱不会被存储。

2. 匿名ID的使用目的

匿名ID仅用于验证付费订阅状态。该ID无法用于识别或追踪个人。

3. 数据存储

所有闹钟数据和位置信息仅存储在用户设备上，不传输至外部服务器。

4. 第三方共享

Ringinout不与任何第三方共享用户信息。

5. 权限说明

  • 位置权限（含后台）：用于检测到达/离开指定地点以触发闹钟。
  • 麦克风权限（RECORD_AUDIO）：仅在您通过语音输入闹钟时使用，录音内容不会被存储或传输。
  • 通知权限：向用户发送闹钟提醒所必需。

6. 联系方式

如有隐私相关问题，请使用应用内的"发送反馈"功能与我们联系。
